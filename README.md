# 📁 05_Clients — Hồ sơ Khách Hàng

Thư mục này lưu trữ hồ sơ đầy đủ của các khách hàng đã và đang được tư vấn triển khai.

## Cách sử dụng

1. **Tạo khách hàng mới:** Copy thư mục `_Template/` và đổi tên thành `[Ten_Cong_Ty]/`.
2. **Điền thông tin:** Bắt đầu với `00_Overview.md`, sau đó điền `01_Problem_Statement.md`.
3. **Cập nhật liên tục:** Ghi lại trao đổi vào `04_Communication_Log.md` sau mỗi buổi gặp.
4. **Lưu tài liệu:** Đặt file đính kèm (PDF, Excel, ảnh) vào thư mục `05_References/`.

## Cấu trúc một hồ sơ khách hàng

```
[Ten_Cong_Ty]/
├── 00_Overview.md          ← Thông tin tổng quan + trạng thái dự án
├── 01_Problem_Statement.md ← Đề bài chi tiết (AS-IS / TO-BE)
├── 02_BA_Report.md         ← Kết quả phân tích BA (copy từ Tpl_BA_Report)
├── 03_SA_Report.md         ← Kết quả SA (copy từ Tpl_SA_Report)
├── 04_Communication_Log.md ← Inbox trao đổi theo timeline
└── 05_References/          ← Tài liệu đính kèm, link, file
```
